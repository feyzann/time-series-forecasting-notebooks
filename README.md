This repository contains Jupyter notebooks for synthetic time series data generation
and forecasting using classical machine learning and deep learning models.

## Notebooks

- `dataset.ipynb`  
  Synthetic time series data generation (AR(1) noise, trend, monthly/weekly seasonality)

- `mlmodels.ipynb`  
  Classical machine learning models for time series forecasting

- `dlmodels.ipynb`  
  Deep learning models for time series forecasting
